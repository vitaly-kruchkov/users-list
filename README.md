# A user list with search and pagination options.

[Demo](https://users-list-wheat.vercel.app/)

## ⚙️ Local Development

### 1. Clone the repository

Prerequisites
• Node.js >= 20
• pnpm / npm / yarn

```bash
git https://github.com/vitaly-kruchkov/users-list
cd users-list
```

### 2. Install dependencies

```bash
npm install
```

### 3. Deployment

```bash
npm run dev
```

### 4. Technologies

```
• React
• TypeScript
```
